# Trip Viz (alpha)

Visualizing trip data with `Mapbox GL JS`, `react-map-gl` and `deck.gl`.

To run the app locally follow these steps:

* clone repo
* `npm install`
* add `.env.local` file in the root of the project with `REACT_APP_MAPBOX_TOKEN='{your-mapbox-token}'`
* `npm start`
